# Simple Banking Application

This is a basic command-line banking application implemented in C++. It allows users to create accounts, perform transactions such as deposits and withdrawals, and manage their account information.

## Features

- Create a new bank account with account number, name, mobile number, and address.
- Perform transactions: check balance, deposit, withdrawal.
- Update account information: mobile number and address.
- Simple command-line user interface.

## Getting Started

1. Make sure you have a C++ compiler installed on your system.
2. Download or clone this repository to your local machine.
3. Navigate to the project directory in your terminal.

```bash
cd path/to/simple-banking-app
```

4. Compile the code using the C++ compiler.

```bash
g++ -o bankingApp main.cpp
```

5. Run the compiled program.

```bash
./bankingApp
```

1. When you run the program, you will be presented with a menu:

   - **Enter your Account**: Lets you access an existing account to perform transactions and manage account details.

2. Follow the on-screen instructions to interact with the application.
3. You can perform transactions such as checking balance, depositing funds, withdrawing funds, and updating account details.
